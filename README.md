# Persona & Professional Hub 🚀

> **HR 전문가의 포트폴리오와 기업의 소통 문화가 만나는 공간입니다.**

이 프로젝트는 개인의 커리어 자산인 '개인 홈페이지' 내에 사내 임직원을 위한 '명예의 전당' 및 'HR 챗봇' 기능을 통합한 하이브리드 플랫폼입니다.

## 🌟 핵심 메뉴 (Core Sections)

1. **[Honor] 명예의 전당**
   - 퇴임하시는 사장님 및 실장님을 위한 헌정 페이지.
   - 2년 주기의 보직 변경에 대응하는 유연한 템플릿 구조.
   - 실시간 롤링페이퍼 및 기념 영상(YouTube) 임베딩.

2. **[Service] HR 헬퍼**
   - 직원들이 가장 궁금해하는 **후생복지** 관련 질문에 자동 응답하는 챗봇.
   - "올해 콘도 신청은 언제인가요?", "의료비 지원 범위는?" 등의 질문에 즉시 대응.

3. **[Personal] 프로필 & 블로그**
   - 관리자(최성진)의 전문 경력 및 조직문화 개선 사례 공유.

## 🛠 기술 스택 (Tech Stack)

- **UI Framework**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: FontAwesome 6.4.0
- **Animations**: CSS Keyframes (Fade-in, Confetti)
- **Interactive**: JavaScript (Tab switching, BGM Player)

## 📂 프로젝트 구조

- `/index.html`: 전체 사이트의 메인 대시보드
- `/honor-template.html`: 보직자 퇴임/취임 기념 표준 템플릿
- `/assets/`: 이미지, 오디오 파일 및 CSS 라이브러리
- `/scripts/`: 챗봇 로직 및 데이터 핸들링 스크립트
